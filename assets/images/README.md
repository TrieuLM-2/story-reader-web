# Images Folder

## Cover Images

Upload story cover images here with the naming format:
- `story-slug.jpg` or `story-slug.png`
- Example: `the-wolf-at-the-door.jpg`

## Recommended Image Specs
- **Dimensions**: 400x600px (2:3 ratio)
- **Format**: JPG or PNG
- **Size**: < 500KB for optimal loading
- **Quality**: 80-90% compression

## Note
Binary image files cannot be uploaded via GitHub API text method. Please upload manually via:
1. GitHub web interface (Add file > Upload files)
2. Git command line
3. GitHub Desktop
