# Góc Truyện Của Tui

## Mô tả
Website đọc truyện trực tuyến với giao diện thân thiện và hiện đại.

## Tính năng
- Hiển thị danh sách truyện với ảnh bìa
- Phân loại theo thể loại
- Giao diện responsive, tương thích mobile
- Hiệu ứng hover mượt mà

## Công nghệ
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## Cách sử dụng
1. Clone repository này
2. Mở file `index.html` trên trình duyệt
3. Hoặc deploy lên GitHub Pages
